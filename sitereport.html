Introduction

A responsive portfolio website is a website in which you can add your skills or company details. We can add our Personal details, skills, experiences. In the website we can add our design, sketches and photos in the website.

Objective of the Assignment

The main objective of the assignment is to learn to make a responsive website by using HTML and CSS. In this assignment, we will learn to add pictures to our website, learn to make contact forms, learn to write blogs, learn to give CSS, learn to make pages responsive and attractive.

Assignment Experience
